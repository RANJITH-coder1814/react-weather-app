# react-weather-app
A simple Weather Application built with React JS that fetches real-time weather data using the OpenWeather API.

A simple Weather App built using React JS and OpenWeather API.

## Features
- Search weather by city
- Shows temperature, humidity, wind speed
- Error handling for invalid city
- Responsive UI

## Tech Stack
- React JS
- OpenWeather API
- CSS

## Setup

1. Clone repository
2. Run `npm install`
3. Add your OpenWeather API key inside App.js
4. Run `npm start`
